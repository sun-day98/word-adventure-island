# AI-Writer 使用指南

## 🤖 功能介绍

AI-Writer是基于RWKV-LM语言模型的中文小说生成工具，由BlinkDL开发，专注于自动化续写网络小说。

## ✨ 主要特性

### 🎭 支持的网络小说题材

| 题材 | 描述 | 常见元素 |
|------|------|----------|
| 玄幻 | 修仙魔法题材 | 战斗场面、升级系统、神秘宝物、师父传授 |
| 言情 | 情感恋爱题材 | 情感纠葛、误会化解、浪漫告白、甜蜜互动 |
| 都市 | 现代都市题材 | 职场竞争、人际关系、都市情感、事业奋斗 |
| 科幻 | 未来科技题材 | 科技设定、未来社会、太空探索、AI意识 |
| 历史 | 古代历史题材 | 历史背景、古代礼仪、江湖恩怨、宫廷斗争 |

### 🛠️ 生成模式

| 模式 | 功能 | 适用场景 |
|------|------|----------|
| 自动续写 | 基于上文内容自然续写 | 写作中断、情节发展 |
| 灵感创作 | 生成富有想象力的创意内容 | 开头创作、创意枯竭 |
| 对话生成 | 生成符合题材的精彩对话 | 人物互动、情节推进 |
| 场景描写 | 生成生动的场景描述 | 环境设定、氛围营造 |
| 高潮情节 | 生成紧张刺激的情节 | 故事高潮、冲突激化 |

## 🚀 使用方法

### 1. 启动AI-Writer

在AI写作助手的右侧面板中，点击"启动AI小说生成器"按钮。

### 2. 选择生成参数

- **题材选择**: 从下拉菜单选择想要的小说题材
- **生成模式**: 根据创作需求选择合适的生成模式

### 3. 生成内容

点击"生成内容"按钮，AI-Writer将根据您的选择生成相应的小说内容。

### 4. 应用到编辑器

生成完成后，可以：
- 点击"应用到编辑器"将内容插入到编辑器中
- 继续编辑或再次生成其他内容

## 📋 使用技巧

### ✅ 最佳实践

1. **提供上下文**: 在编辑器中输入一些文字，AI-Writer会基于上下文生成更连贯的内容
2. **尝试不同模式**: 根据创作阶段选择合适的生成模式
3. **多次生成**: 如果不满意第一次生成结果，可以多次尝试
4. **结合编辑**: 将AI生成的内容作为灵感，进行个性化修改

### ⚠️ 注意事项

1. **娱乐优先**: 生成内容主要基于网络小说数据，可能缺乏现实常识
2. **质量评估**: AI生成内容仅供参考，需要人工审核和修改
3. **版权意识**: 使用生成内容创作时请注意版权和原创性
4. **逐步完善**: 可以通过多次生成和编辑来完善作品

## 🔧 技术参数

- **模型**: RWKV-LM v5
- **最大生成长度**: 512字符
- **支持语言**: 中文
- **生成速度**: 1-4秒
- **质量评估**: 自动评分系统
