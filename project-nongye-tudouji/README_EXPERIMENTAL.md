```bash

```

### 🌐 Web控制界面

1. **启动Web服务**：

2. **访问控制界面**：

## 🔧 故障排除

### 常见问题

#### Q: 树莓派无法启动
**解决方案**：
1. 检查SD卡是否正确烧录
2. 确认电源输出≥5V/2A
3. 检查HDMI连接

#### Q: 电机不转动
**解决方案**：
1. 检查GPIO接线
2. 确认电机电源充足
3. 运行电机测试代码

#### Q: 摄像头无法工作
**解决方案**：
```bash
sudo raspi-config
# Interface Options → Camera → Enable
sudo reboot
```

#### Q: WiFi连接问题
**解决方案**：
1. 检查WiFi配置文件
2. 确认信号强度
3. 重启网络服务
