## 🔧 使用说明

### 启动方式
1. **自动启动**：双击 `启动AI写作助手.bat`
2. **网络良好**：自动启动现代化界面
3. **网络受限**：自动启动离线版界面

### 🎯 推荐使用现代化版
- 更专业的界面设计
- 更强大的AI功能
- 更流畅的用户体验
- 完全离线运行

### 📝 基础操作
1. **新建文档**：点击"新建文档"或按Ctrl+N
2. **智能写作**：在编辑区输入，AI会实时提供建议
3. **快速续写**：按Ctrl+K触发AI续写
4. **即时润色**：按Ctrl+P进行文本优化
5. **自动保存**：文档会自动保存到本地存储

### 🤖 AI高级功能
1. **体裁识别**：自动识别文本体裁并提供相应建议
2. **智能续写**：根据上下文智能续写内容
3. **文本分析**：分析写作风格、情感语调等
4. **大纲生成**：一键生成专业的写作大纲
5. **语法检查**：实时检查并修正语法错误

### 📚 AI拆书功能（专用界面推荐）
1. **书籍分析**：粘贴书籍内容进行深度结构分析
2. **核心提取**：自动提取核心观点和实用建议
3. **笔记生成**：生成综合性读书笔记
4. **卡片制作**：创建便于复习的知识卡片
5. **价值评估**：多维度评估书籍价值

### ✍️ AI仿写功能（专用界面推荐）
1. **风格识别**：自动分析原文写作风格特征
2. **模板丰富**：支持文学、现代、网络、专业等风格
3. **仿写生成**：精准模仿目标风格生成内容
4. **质量评估**：评估仿写质量并提供改进建议
5. **学习路径**：提供系统的风格学习计划

### ⌨️ 快捷键
- `Ctrl+S` - 快速保存
- `Ctrl+K` - AI智能续写
- `Ctrl+P` - 文本润色优化
- `Ctrl+N` - 新建文档
- `Ctrl+D` - 快速拆书
- `Ctrl+I` - 快速仿写
- `F11` - 全屏模式
- `Esc` - 退出全屏/专注模式

## 💡 故障排除

### 🚀 现代化版问题
1. **界面空白**：清除浏览器缓存，刷新页面
2. **AI功能无响应**：检查JavaScript是否启用
3. **数据丢失**：检查本地存储权限

### 🌐 传统版问题
1. **界面无法显示**：检查网络连接，关闭防火墙
2. **样式加载失败**：使用现代化版作为替代
3. **资源加载慢**：等待CDN加载或切换到离线版

### 📱 通用问题
1. **文档无法保存**：检查浏览器本地存储权限
2. **快捷键无效**：确保浏览器获得焦点
3. **字体显示异常**：使用Chrome或Edge浏览器

### 🔧 最佳实践
1. **推荐浏览器**：Chrome、Edge、Firefox最新版
2. **定期备份**：重要文档建议导出备份
3. **清理缓存**：定期清理浏览器缓存
4. **更新版本**：及时更新到最新版本
