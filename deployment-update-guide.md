## 🛠️ 故障排除

### 常见问题

#### 1. 语音功能不工作
**解决方案**:
- 检查浏览器是否支持Web Speech API
- 确保在HTTPS环境下访问
- 检查浏览器麦克风权限

#### 2. 机器人不显示
**解决方案**:
- 刷新页面 (Ctrl+F5)
- 清除浏览器缓存
- 检查网络连接

#### 3. 样式显示异常
**解决方案**:
- 检查CSS文件是否正确加载
- 尝试不同浏览器
- 检查开发者工具错误信息

#### 4. 功能按钮无响应
**解决方案**:
- 检查JavaScript是否正常执行
- 查看浏览器控制台错误
- 确保所有文件都已加载
