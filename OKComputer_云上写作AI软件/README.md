# AI写作助手

一个功能完整的AI写作软件网页应用，提供智能写作辅助、文档管理、写作统计等核心功能。

## 功能特性

### 🚀 核心功能
- **智能写作助手** - 实时AI写作建议和内容生成
- **文档管理系统** - 项目分类、版本控制和协作功能
- **写作统计分析** - 详细的写作数据和进度追踪
- **创意激发工具** - 随机主题生成和情节发展建议

### 🎨 设计特色
- **现代简约界面** - 专业且富有创意的设计风格
- **流畅动画效果** - 使用Anime.js和p5.js实现丰富的交互
- **响应式布局** - 完美适配桌面、平板和移动设备
- **个性化主题** - 支持自定义颜色和界面设置

### 🛠️ 技术栈
- **前端框架**: HTML5 + CSS3 + JavaScript ES6+
- **样式框架**: Tailwind CSS
- **动画库**: Anime.js, p5.js
- **数据可视化**: ECharts.js
- **轮播组件**: Splide.js
- **字体**: Noto Sans SC, Noto Serif SC

## 页面结构

### 1. 主页面 (index.html)
- 写作编辑器界面
- 左侧项目文档管理
- 中央富文本编辑区域
- 右侧AI助手面板
- 实时写作统计和建议

### 2. 仪表板 (dashboard.html)
- 写作数据概览
- 交互式统计图表
- 项目进度管理
- 写作目标设定

### 3. 模板库 (templates.html)
- 多种写作模板
- 分类筛选和搜索
- 模板预览功能
- 一键应用模板

### 4. 设置页面 (settings.html)
- 个人资料管理
- 写作偏好设置
- AI助手配置
- 界面自定义选项

## 使用方法

### 本地运行
1. 下载所有文件到本地目录
2. 使用HTTP服务器运行（如：`python -m http.server 8000`）
3. 在浏览器中访问 `http://localhost:8000`

### 功能使用
1. **开始写作**
   - 点击"新建文档"创建新的写作项目
   - 选择文档类型和写作风格
   - 在编辑器中开始写作

2. **AI辅助功能**
   - AI实时提供写作建议和优化
   - 使用"AI续写"功能自动生成内容
   - 获取创意灵感和情节发展建议

3. **文档管理**
   - 在左侧边栏管理项目和文档
   - 查看写作统计和进度
   - 导出文档为多种格式

4. **数据分析**
   - 在仪表板查看详细的写作数据
   - 分析写作习惯和效率
   - 设定和追踪写作目标

## 特色功能

### AI智能写作
- **实时建议** - 根据上下文提供词汇和句式建议
- **内容生成** - 基于关键词自动生成段落内容
- **风格调整** - 支持多种写作风格切换
- **语法检查** - 自动检测和修正语法错误

### 数据可视化
- **写作趋势** - 展示写作字数和时间趋势
- **项目分布** - 可视化项目类型分布
- **效率分析** - 分析最佳写作时间和效率
- **进度追踪** - 实时显示写作目标完成度

### 个性化体验
- **主题定制** - 多种颜色和界面主题选择
- **字体设置** - 支持字体大小和样式调整
- **通知管理** - 自定义提醒和通知设置
- **数据同步** - 支持云端数据同步
