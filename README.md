# 单词冒险岛 🏝️

一款专为小学生设计的英语单词学习游戏，将单词记忆与冒险故事完美融合，让学习变成一场精彩的探索之旅！

## 🌟 游戏特色

### 📚 核心理念
- **ARCS动机模型**：从注意、相关性、自信心、满足四个维度激发学习动机
- **情境记忆法**：每个单词都有生动的记忆口诀和联想场景
- **游戏化学习**：通过冒险剧情让单词学习充满乐趣

### 🗺️ 完整世界观
- **主线剧情**：小学生霏霏与伙伴拯救被"遗忘迷雾"笼罩的童话世界
- **角色成长**：每个角色都有独特的成长弧光和深刻的故事
- **分级岛屿**：根据年级和难度设计的主题岛屿

### 🎮 多样化挑战
- **发音挑战**：语音识别跟读，提升发音准确度
- **拼写防御**：正确拼写单词抵御怪物进攻
- **单词配对**：连接单词与释义的训练
- **句子构建**：用单词创造正确的句子
- **极速挑战**：限时答题考验反应速度
- **综合大师**：全方位能力测试

## 🚀 快速开始

### 环境要求
- 现代浏览器（Chrome 80+, Safari 13+, Firefox 75+）
- 支持响应式设计，适配手机和平板

1. 克隆项目到本地
2. 直接在浏览器中打开 `index.html`
3. 开始你的单词冒险之旅！

## 🎯 核心功能

### 🏝️ 岛屿系统
- **奇妙生物岛**：适合1-2年级，学习基础名词和动词
- **能量火山岛**：适合3-4年级，学习行动和激励类词汇
- **智慧金字塔**：适合5-6年级，学习抽象概念和复杂表达

### 👥 角色系统
- **霏霏**：玩家主角，从自卑到自信的成长故事
- **丘比**：引导者，从教条到开放的转变
- **雷**：大魔法师之子，从个人主义到团队精神的成长
- **邓彼**：反派角色，增加故事的冲突性

### 📊 学习系统
- **艾宾浩斯遗忘曲线**：科学的复习时间安排
- **动态难度调整**：根据学生表现智能推荐挑战
- **多感官刺激**：视觉、听觉、触觉全方位学习体验

### 🏆 成就系统
- **学习成就**：记录每个学习里程碑
- **挑战成就**：奖励游戏技能提升
- **社交成就**：鼓励良性竞争和合作

## 🔧 开发指南

### 添加新单词
```javascript
// 在 word-adventure-data.js 中添加
{
    id: 'new_word',
    word: 'example',
    chinese: '例子',
    pronunciation: '/ɪɡˈzæmpl/',
    mnemonic: '俺格赞破，这是一个好例子',
   联想: '课堂上老师举例子帮助理解',
    grade: 3,
    category: 'concept',
    difficulty: 'intermediate'
}
```

### 创建新关卡
```javascript
// 在 LevelConfig 中添加新关卡
{
    id: 'new_level',
    name: '关卡名称',
    words: ['word1', 'word2', 'word3'],
    challenges: ['pronunciation', 'spelling'],
    rewards: { exp: 100, coins: 50 }
}
```

### 自定义挑战类型
```javascript
// 在 ChallengeTypes 中添加新类型
newChallenge: {
    id: 'new_challenge',
    name: '新挑战类型',
    description: '挑战描述',
    icon: '🎯',
    duration: 120
}
```
